Installation

Install Complete Metronome

The browser version is fully functional - you don't have to install it.

But if you want to, chose the setup that fits your device to install as an app.

Android

Download the APK from GitHub Releases:

https://github.com/mojasp/CompleteMetronome

You can also install it as a PWA, but it will not run in the background.

Links to Play Store and F-Droid will follow.

iOS (Safari)

Turn off Auto-Lock due to iOS restrictions on PWAs: Settings → Display & Brightness → Auto-Lock → Never. Alternatively, keep using the browser-based version.

  1. Open this site in Safari.
  2. Tap Share → Add to Home Screen.
  3. Name it and tap Add.

This installs the PWA on your home screen.

An iOS app is unlikely, since i do not have access to the Apple Developer Program membership and don't want to pay 100 Euros for it.

This software is free and open source and will never have ads.
